# Quantum, Honestly — Course Notebooks

Runnable Python companions to the **Quantum, Honestly** YouTube series — a math-honest,
build-it-from-the-ground-up tour of quantum mechanics for the curious non-physicist.

Each notebook is the live-code half of one episode: the same worked examples done on screen,
powered by a tiny shared toolkit (`qm_toolkit.py`) written in plain NumPy/SciPy. Every figure —
Bloch spheres, measurement histograms, interference plots — is generated by the code, not pre-baked.

> ▶ **Watch the series:** [**Quantum, Honestly** — full playlist on YouTube](https://youtube.com/playlist?list=PLQOOPoCjGTwk)

## Run them

**Zero install — Google Colab:** click a badge above. The notebook's first cell auto-fetches
`qm_toolkit.py`, so everything below it just runs.

**Zero install — Binder:** [![Binder](https://mybinder.org/badge_logo.svg)](https://mybinder.org/v2/gh/HogRed/quantum_honestly_public/main?labpath=episodes)

**Locally:**

```bash
git clone https://github.com/HogRed/quantum_honestly_public.git
cd quantum_honestly_public
pip install -r requirements.txt
jupyter lab
```

Then open any `episodes/E*/E*_lecture.ipynb` and run top to bottom.

## What's inside

```
qm_toolkit.py              the whole course's physics in one file (NumPy/SciPy)
requirements.txt
episodes/
  E1-two-slit/             E1_lecture.ipynb  +  media/  (animation clips)
  E2-quantum-coin/         E2_lecture.ipynb  +  media/
  E3-measurement/          E3_lecture.ipynb  +  media/
```

`qm_toolkit.py` is the one import every notebook makes (`import qm_toolkit as qm`): state vectors,
the Born rule (`born_probs`, `measure`, `born_sample`), overlaps, and Bloch-sphere plotting. The
notebooks build all their figures from it live.

The short clips embedded in the notebooks are excerpts from the episodes; they play when you run a
notebook in Jupyter or Binder. (GitHub's static preview and Colab may not render the embedded video —
that's expected; the code cells still run everywhere.)
